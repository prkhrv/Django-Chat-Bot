# Django Chat Bot
 A chat Bot made Using Django
 
 ## Install Dependencies
 ```
 pip install -r requirements.txt
 ```
 
 ## Run
 ```
 python manage.py runserver
 ```
 
 ## Migrate
 
 ```
 python manage.py migrate --run-syncdb
 ```
 
 ## Create Super User
 
 ```
 python manage.py createsuperuser
 ```
